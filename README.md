# Medicine Package Classification using Deep Learning

## Overview
This project focuses on automated medicine package classification using Computer Vision and Deep Learning techniques. The objective is to explore AI-based quality control solutions for identifying and classifying medicine packages.

## Models Implemented
- Custom Convolutional Neural Network (CNN)
- VGG16
- ResNet50
- DenseNet

Multiple deep learning architectures were trained and compared to identify the model with better classification performance.

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook

## Workflow
1. Image preprocessing
2. Model training
3. Performance evaluation
4. Comparison of different CNN architectures
5. Prediction and classification

## Project Highlights
- Implemented transfer learning using pretrained models
- Compared multiple architectures for accuracy improvement
- Applied deep learning techniques for pharmaceutical quality control automation
